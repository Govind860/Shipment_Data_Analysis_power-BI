# 📦 Shipment Data Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Shipment Data Analysis Dashboard** built using Power BI. It helps analyze logistics performance, delivery efficiency, and shipment trends to support better business decisions.

---

## 🎯 Objectives

* Analyze shipment and delivery performance
* Track on-time vs delayed deliveries
* Identify region-wise shipment distribution
* Evaluate shipping modes and efficiency

---

## 📂 Dataset

The dataset used in this project contains shipment-related information such as:

* Order ID
* Shipment Date
* Delivery Date
* Region / Location
* Shipping Mode
* Delivery Status (Delivered / Delayed / Pending)
* Quantity / Sales (if applicable)

---

## 📊 Dashboard Features

### 🔹 Key Performance Indicators (KPIs)

* Total Shipments
* On-Time Deliveries
* Delayed Shipments

### 🔹 Visualizations

* 📈 Shipment Trends Over Time (Line Chart)
* 📊 Region-wise Shipment Analysis (Bar Chart)
* 🥧 Delivery Status Distribution (Pie Chart)
* 🚚 Shipping Mode Comparison

### 🔹 Filters (Slicers)

* Region
* Shipment Date
* Delivery Status
